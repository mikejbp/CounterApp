# Counter App / App Contador

## Description / Descripción: 
*Simple counter app with responsive design which has decrement, increment and reset buttons.*
<br><br>
*Simple aplicación contador con diseño responsivo, la cual posee botones de decrementar, incrementar y refrescar.*
